# ansible-manage-lvm

Ansible role to manage LVM Groups/Logical Volumes.

> NOTE: Can be used to create, extend or resize LVM Groups and volumes.

## Requirements

Devices/disks to be members of the LVM setup **must be** identified prior to
using this role.
